# Mobile-Sales-Analysis-PowerBI
Power BI project analyzing mobile sales trends, customer behavior, and business recommendations.

## Project Overview

This project analyzes mobile phone sales data using Power BI to identify sales trends, customer behavior, regional performance, and business opportunities.

## Business Objective

To uncover actionable insights that can help improve sales performance, optimize inventory allocation, and support data-driven decision making.

## Tools Used

- Power BI
- Excel
- Data Analysis
- Data Visualization

## Key Insights

- Sales peaked in July and were lowest in September.
- Saturday generated the highest revenue.
- Delhi, Hyderabad, and Mumbai were top-performing markets.
- Customer satisfaction levels were generally high.
- Payment methods were evenly distributed.

## Business Recommendations

- Increase marketing efforts during low-performing months.
- Focus inventory on high-performing regions.
- Launch weekend-focused campaigns.
- Expand into Tier-2 markets.
- Introduce loyalty and referral programs.
